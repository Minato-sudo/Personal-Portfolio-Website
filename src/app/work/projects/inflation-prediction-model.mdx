## Overview

This project focused on forecasting inflation using a comparative modeling approach instead of relying on a single technique. Talha used R to evaluate how classical time-series and machine learning models behaved across broader economic patterns and correlated indicators.

## Modeling Approach

- Built ARIMA baselines for trend-aware inflation forecasting.
- Compared performance against SVM and Random Forest models.
- Examined cross-variable relationships to understand which macroeconomic features carried the strongest predictive value.

## Why It Matters

By comparing multiple model families in one pipeline, the study highlighted the trade-offs between interpretability, flexibility, and forecast accuracy. That made the project useful both as an academic exercise and as a practical forecasting workflow.

## Stack

- R
- ARIMA
- SVM
- Random Forest
- Statistical Modeling

## Outcome

The final system demonstrated a structured way to compare model behavior against real-world economic data and created a strong foundation for extending the work into richer macroeconomic forecasting.
